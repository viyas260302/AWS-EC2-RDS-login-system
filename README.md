# AWS-EC2-RDS-login-system
A PHP Login &amp; Registration System deployed on AWS EC2 with Amazon RDS (MySQL), Apache2, and Linux.

##  Project Overview

This project demonstrates how to deploy a PHP Login & Registration application on AWS using Amazon EC2 and Amazon RDS (MySQL).

Users can register with a username and password and log in through a web interface. User information is stored in an Amazon RDS MySQL database.

## Features

- User Registration
- User Login
- PHP & MySQL Integration
- Amazon EC2 Deployment
- Amazon RDS Database
- Apache2 Web Server
- Linux (Ubuntu)

## Technologies Used

- Amazon EC2
- Amazon RDS (MySQL)
- PHP
- Apache2
- MySQL
- Linux (Ubuntu)
- AWS Security Groups
  
## Setup Steps

1. Launch an EC2 Ubuntu instance.
2. Install Apache2, PHP, and MySQL client.
3. Create an Amazon RDS MySQL database.
4. Configure Security Groups for EC2 and RDS.
5. Connect EC2 to RDS.
6. Create the LoginDB database and users table.
7. Deploy the PHP application.
8. Access the application using the EC2 Public IP.
